Why loops

为什么必须循环：先防方向错，再防执行漂移。

这不是把 Prompt 写得更长，而是把“方案证伪”和“交付验收”拆成两个独立闭环。每次循环都必须留下文件交接物，下一层只读取这些交接物继续推进。

Loop 1 解决：怎么做才是对的010-040 把目标、风险、证据、路线、反证、裁决和 Spike 都落成文档，避免 Agent 在错误方向上高效执行。
Loop 2 解决：怎样把已选路线做对Claude Code 负责 plan/Test Oracle/终检，Codex 只能按 approved plan 执行，避免执行时偷换范围、弱化测试或漏写复盘。
文件是交接总线，不是聊天记录Spec 用 010-040；模块交付用 NNN 同号调研、plan、复盘、review；证据 ID 用 L1-E / CC-R / CX-R / CC-V。
